BhoomiTech Apps

BhoomiTech Apps is a live repository of applications developed by our members to share and support various activities of the organization and also allow free use by the public.

Sl Name and Url Description
1 Geo Image
Open Application 		This application displays geotagged images on a interactive interface of a map and list of images. Images and json files must be uploaded to specific folders in the root directory.
2 GeoJson Attribute Editor
Open Application 		This application provides a friendly interface to edit and save attributes of a GeoJson file. The application handles files in local disk only.
3 GeoJson Maker
Open Application 		This application is a map maker. It allows drawing points, polylines and polygons and save as GeoJsons. Application handles files in local disk only.
4 Map Viewer
Open Application 		Map Viewer is a simple GIS viewer which provides
  • Multilayer map view
  • Layer styling
  • Pop-Up Manager
  • Navigation and Routing
  • Search Data & Geocode Database
  • Route Speed Control
  • Heat Map, Clustering and more...
You can load multiple layers of GeoJson or KMZ datasets. Application handles local files stored to local disk only.
